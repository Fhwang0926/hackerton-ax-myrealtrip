# 01. Problem Definition

# 공개 데이터 기반 문제 정의

## 과제 목표

'공개·검증 가능한 문제'를 정의하고 이를 해결하는 Codex Plugin을 구현한다.

---

# 해결하려는 문제

마이리얼트립 마케팅팀과 MD는 매주 아래 업무를 반복한다.

1. 어떤 상품을 이번 주에 밀어야 하는가?
2. 왜 그 상품을 선택해야 하는가?
3. 어떤 채널(Instagram/Facebook/네이버)에 어떤 형태로 홍보해야 하는가?
4. 캠페인 초안을 어떻게 작성할 것인가?

현재 이 과정은 다양한 공개 데이터를 사람이 직접 조사하여 수행한다.

---

# 문제를 증명하는 방법

매주 아래 질문에 답한다.

- 최근 많이 홍보한 여행지는?
- 최근 거의 홍보하지 않은 여행지는?
- 외부 트렌드와 현재 캠페인이 일치하는가?
- 시즌성과 맞는가?
- 경쟁력 있는 프로모션인가?

---

# 플러그인이 해결하는 범위

공개 데이터

1. Weekly Trend

4. SNS 초안
